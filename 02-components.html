Drop-in UI

If you want even more of the framework, there are already some ready-to-use components, that will continue to evolve and become more.

Navigation Bar XX KBModules: ButtonsJavaScript: Collapse

The current implementation is really basic and just allows to display a navigational bar that stretches to the whole available space. But it will become more useful in the future. Here is a full-blown example:

Brand name
  • Lorem.
  • Recusandae.
  • Est.
  • Possimus.
  • Quod?
  • Saepe.
<div class="navbar collapsed" role="navigation" id="demoNavbar">
  <div class="navbar-header">
    <a href="#" class="navbar-brand">Brand name</a>
    <button type="button" class="navbar-toggle" aria-label="Toggle Menu" data-target="#demoNavbar" data-action="collapse"><span class="icon"></span></button>
  </div>
  <div class="navbar-collapse">
    <ul class="navbar-nav">
      <li><a href="#">Lorem.</a></li>
      <li><a href="#">Recusandae.</a></li>
      <li class="active"><a href="#">Est.</a></li>
      <li><a href="#">Possimus.</a></li>
      <li><a href="#">Quod?</a></li>
      <li><a href="#">Saepe.</a></li>
    </ul>
  </div>
</div>

The basic navigation bar is just a block-level container with the navbar class. To make it collapsable if the viewport gets to small, just wrap the collapsable content in a .navbar-collapse container and the content, that should remain visible in a .navbar-header container. Use the .navbar-toggle button to allow expansion on JavaScript enabled devices. Use the navbar-nav class for menus for the navigation inside the bar. The navbar-brand class allows prominent display of an element.
