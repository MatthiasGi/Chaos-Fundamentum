JavaScript

Additional fanciness!

For modular interactivity, Chaos-Fundamentum does also provide optional JavaScript-Plugins, that also provide fallback for non JavaScript enabled devices. Check out the features below!

Each module will apply a class to the body on successful load. This allows you to provide fallback, if the load fails. The added class is displayed beside the module name.

Collapse.c-collapse

This module allows the creation of collapsable content (for example the navigation bar). It searches for all elements with a data-action="collapse" attribute. The data-target attribute should contain a query slector (e.g. data-target="#navbar"). The plugin will apply a click-listener to the element and toggle the collapsed and expanded classes on the target(s). It will ensure that on load only one of the two is applied as default to the target.
