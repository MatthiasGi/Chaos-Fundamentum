Chaos Fundamentum

Chaos Fundamentum is a modular, responsive frontend-framework, that is heavily inspired by other frameworks, such as Twitter Bootstrap. However it does provide it's own set of functionality, carefully hand-crafted and as minimal as possible, putting the developer (or designer) and his wishes first.

The framework is intended to work with the following browsers:

  • Internet Explorer 10+
  • Firefox 24+
  • Chrome 21+
  • Safari 5.1+
  • Opera 12.1+
  • iOS Safari 5.1+
  • Android Browser 4+

Please keep in mind, that it uses the most modern standard and such isn't viewable by approximately 25 % of the users out there (see "Can I use" usage table).

The following site showcases the usage of the framework and provides a few examples. Please begin by exploring the site with the menu above.
