Chaos Fundamentum

Chaos Fundamentum is a modular, responsive framework, that is heavily inspired by other frameworks, such as Twitter Bootstrap. However it does provide it's own set of functionality, carefully crafted and as minimal as possible, putting the developer and his wishes first.

The framework is intended to work with the following browsers:

  • Internet Explorer 10+
  • Firefox 24+
  • Chrome 21+
  • Safari 5.1+
  • Opera 12.1+
  • iOS Safari 5.1+
  • Android Browser 4+

Please keep in mind, that it uses the most modern standard and such isn't viewable by approximately 25 % of the users out there (see "Can I use" usage table).

Modules

On top of the most basic features, Chaos Fundamentum provides completly optional modules that will ease the fast developing and prototyping of homepages and webapps. And if you don't need a module for productive use: Just don't compile it.

Simple grid system

The simple grid system allows you to align cells inside a single row (no fancy multiline grid is provided at the moment). It uses the modern and futureproof flexbox-model. As such, there's no real fallback for old browsers possible (however they will render the content stacked vertically).

Nesting columns and auto-computing is easy.
